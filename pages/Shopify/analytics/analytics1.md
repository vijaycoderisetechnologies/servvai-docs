# Analytics Documentation

## Accessing the Analytics Section
To open the Analytics section:
- Go to the **servv AI** plugin in your WordPress dashboard.
- Click on **Analytics** in the side menu bar in servv ai.

<img src="{{ site.baseurl }}/assets/images/analyticssection.png" alt="Servv" width="200" style="max-width: 100%; height: auto;" />

The **Analytics** section contains the following tabs:

- **Revenue**
- **Resistant**
- **Event**
- **Filter**

## Revenue Tab
- **Default View:** Displays **all revenues**.
- **Filtering Revenues:** Use the **Date Picker** to select a date range.
- After selecting the range, the system will display the revenue between the chosen dates.

<img src="{{ site.baseurl }}/assets/images/analytic1.png" alt="Servv" width="600" style="max-width: 100%; height: auto;" />

## Registrant Tab
- **Default View:** Shows **all resistants**.
- **Filtering by Month and Year:** Two dropdown menus are available:
       - **Month Selector**
       - **Year Selector**
  - Select a specific month or year to view resistants for that period.      
- A **Reset** button is provided to clear the selection.


  <img src="{{ site.baseurl }}/assets/images/analytic2.png" alt="Servv" width="600" style="max-width: 100%; height: auto;" />

## Event Tab
Shows the **total number** of:

- **Active Events**
- **Events Happened**
- **Cancelled Events**

<img src="{{ site.baseurl }}/assets/images/analytic3.png" alt="Servv" width="600" style="max-width: 100%; height: auto;" />

## Filter Tab
Displays the count of filters created in the system:

- **Language Filters** Shows which languages are created, e.g., *English, Hindi*, or any other language.
- **Location Filters** Shows location-based filter options, e.g., *Canada, Hong Kong, India*.
- **Category Filters** Displays available category-based filter options (based on event or system setup).

<img src="{{ site.baseurl }}/assets/images/analytic4.png" alt="Servv" width="600" style="max-width: 100%; height: auto;" />
