# Add Location Filter

### Access the Filter Dashboard
From your **Servv AI** side bar, go to Filters. This section manages all available filter types.

<img src="{{ site.baseurl }}/assets/images/filtersection.png" alt="Servv" width="200" style="max-width: 100%; height: auto;" />

### Click the **Add** button at the top right to begin creating a new filter then select **Location**.

<img src="{{ site.baseurl }}/assets/images/locationfilter.png" alt="Servv" width="300" style="max-width: 100%; height: auto;" />

### Fill Out Location Details and save 

**Location Name** – e.g., "New York City" or "London".

**Location Details** – Optional notes like venue name or address.

**Operational Hours** – Define when the location is active or available.

<img src="{{ site.baseurl }}/assets/images/newlocationform.png" alt="Servv" width="700" style="max-width: 100%; height: auto;" />

The newly created **location filter** will now be available as an option when creating or editing any event.

<img src="{{ site.baseurl }}/assets/images/filter1.png" alt="Servv" width="700" style="max-width: 100%; height: auto;" />
