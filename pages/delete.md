# Delete Event

<iframe width="900" height="500" src="https://www.youtube.com/embed/AJ9j1jg6RkU?si=XFXeOZMwSA99uFKP" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>

### Select **Events** in **Servv AI** to access the Events section, where all events are presented in a table format.

<img src="{{ site.baseurl }}/assets/images/eventsection.png" alt="Servv" width="200" style="max-width: 100%; height: auto;" />

### Identify the event to be deleted within the table. click the **hamburger icon** (three horizontal lines) to display a dropdown menu with the following options:

**Delete Event**: Removes both the main event and all associated recurring events, including past and future occurrences.

<img src="{{ site.baseurl }}/assets/images/deleteevent.png" alt="Servv" width="700" style="max-width: 100%; height: auto;" />

**Delete Occurrence**: Removes only the selected recurring event occurrence, preserving the main event and other events.

- Select **View Occurrences** from the dropdown menu. A list of all recurring event occurrences will appear.
- Locate the specific occurrence you want to delete and click the **hamburger icon** next to it.
- Choose **Delete Occurrence** from the dropdown menu.

<img src="{{ site.baseurl }}/assets/images/deleteevent2.png" alt="Servv" width="700" style="max-width: 100%; height: auto;" />

### After clicking delete or delete occurrences please confirm your action

<img src="{{ site.baseurl }}/assets/images/deleteevent3.png" alt="Servv" width="600" style="max-width: 100%; height: auto;" />
