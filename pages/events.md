# Event Management Overview

Welcome to the Servv AI Event Management guide. This page provides a high-level summary of all core event functionalities supported by the Servv AI plugin for WordPress. Whether you're new or experienced, these sections will help you create and manage events efficiently.

---

## Create an Event

Get started by creating your first event in just a few clicks. With Servv AI, you can configure:

- Event title, date, and time with timezone
- Event type (single, recurring) and location (in-person, virtual or hybrid)
- Speaker/session details
- Ticketing options (paid, free or donation)

[Learn more →](https://support.servv.ai/getting-started/event/createevent/)

 _This is your first step to getting your event live._

---

## View Your Event

Once your event is created, learn how to:

- Preview the event as users would see it
- Access all published and draft events in one place
- Use filters and search to manage large event inventories

[Learn more →](https://support.servv.ai/getting-started/event/view/)

 _Easily find and check the status of your events._

---

## Configure Event Settings

Customize the core settings to align your event with your brand and audience needs:

- Enable/disable RSVP, ticket sales, reminders
- Customize confirmation emails
- Set registration limits, buffer times, and access controls

[Learn more →](https://support.servv.ai/getting-started/event/setting/)

 _Fine-tune your event's behavior and automation settings._

---

## Edit an Existing Event

Update or revise any aspect of your event with ease:

- Modify the schedule, content, speakers, or pricing
- Update venue or access links
- Make changes visible instantly or save as draft

[Learn more →](https://support.servv.ai/getting-started/event/edit/)

 _Keep your event information current and accurate._

---

## Delete an Event

When an event is no longer needed:

- Delete it permanently from your WordPress site
- Understand implications for users and past registrations
- Follow best practices to avoid accidental deletions

[Learn more →](https://support.servv.ai/getting-started/event/delete/)

 _Clean up your event list by removing outdated or test events._
