# Add Language Filter

### Access the Filter Dashboard
From your **Servv AI** side bar, go to Filters. This section manages all available filter types.

<img src="{{ site.baseurl }}/assets/images/filtersection.png" alt="Servv" width="200" style="max-width: 100%; height: auto;" />

### Click the Add button at the top right to begin creating a new filter then select Language.

<img src="{{ site.baseurl }}/assets/images/languagefilter.png" alt="Servv" width="300" style="max-width: 100%; height: auto;" />

### Give Language Name – e.g., English, Spanish, French, etc and save

<img src="{{ site.baseurl }}/assets/images/newlanguageform.png" alt="Servv" width="700" style="max-width: 100%; height: auto;" />

The newly created **Language filter** will now be available as an option when creating or editing any event.

<img src="{{ site.baseurl }}/assets/images/filter1.png" alt="Servv" width="700" style="max-width: 100%; height: auto;" />
