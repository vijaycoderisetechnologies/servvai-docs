General Settings Configuration

This document provides step-by-step instructions for configuring the general settings in the Servv AI WordPress plugin. These settings establish default values that are automatically applied when creating new events, ensuring consistency and efficiency in event setup.

Open General Settings

Click on Settings in servv ai menu in sidebar to open the settings dashboard. In the settings dashboard, click on the General tab to access the general settings configuration panel.

Configuring General Settings

The General Settings panel allows you to set default values for various event parameters. Below is a detailed breakdown of each option:

Time Zone

  • Purpose: Sets the default time zone for all events.
  • How to Configure:
    • Click the dropdown menu labeled Time Zone.
    • Select the desired time zone (e.g., America/Los_Angeles).
  • Outcome: The selected time zone will be applied to all new events created in Servv AI.

Time Format

  • Purpose: Defines the default time format for event times displayed in emails, widgets, and the dashboard.
  • How to Configure:
    • Choose between 12-hour (e.g., 2:00 PM) or 24-hour (e.g., 14:00) format by selecting the appropriate option.
    • Optionally, check the box labeled Hide timezone abbreviation in email, widget, and dashboard to exclude timezone abbreviations (e.g., PDT, EST) from display.
  • Outcome: The chosen format will be used across all event-related displays, with the option to hide timezone abbreviations for cleaner presentation.

Currency Format

  • Purpose: Specifies the default currency for event ticket pricing.
  • How to Configure:
    • Click the dropdown menu labeled Currency Format.
    • Select the desired currency (e.g., USD, EUR).
  • Outcome: The selected currency will be applied to ticket prices for all new events.

Duration

  • Purpose: Sets the default duration for events.
  • How to Configure:
    • Select the desired duration from the dropdown menu (e.g., 1h for 1 hour, 2h for 2 hours).
  • Outcome: New events will automatically adopt the specified duration unless manually overridden.

Start / End Time

  • Purpose: Defines the default start time for events, with the end time calculated based on the default duration.
  • How to Configure:
    • Select the desired Start Time from the provided options.
    • The End Time will be automatically calculated based on the selected start time and the default event duration set above.
  • Outcome: New events will use the specified start time, with the end time adjusted according to the default duration.

Ticket Price

  • Purpose: Sets the default ticket price for events.
  • How to Configure: Enter the desired ticket price in the provided field.
  • Outcome: The entered amount will be set as the default ticket price for all new events.

Saving Changes

After configuring the desired settings, click the Save button at the top of the Settings panel. All configured settings will be saved and applied as default values when creating new events in Servv AI.
