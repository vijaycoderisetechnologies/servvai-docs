# Configure Event Settings

### Accessing Event Settings

!!! info "âš¡ Premium Feature"
    This feature is available only on Premium plans.

- Navigate to the **Events** section in your Servv AI.
- To create a new event, click **New Event**. To edit an existing event, select the event from the list.
- During event creation or editing, locate the **Event Settings** section.

<img src="{{ site.baseurl }}/assets/images/eventsetting1.png" alt="Servv Event Settings" width="500" style="max-width: 100%; height: auto;" />

From here, you can configure visibility and notification options efficiently without navigating complex menus.

**Visibility**

- **Public**: The event is visible to everyone on your site.
- **Unlisted**: The event is private and accessible only via a direct link.

<img src="{{ site.baseurl }}/assets/images/eventsetting2.png" alt="Servv Event Settings" width="600" style="max-width: 100%; height: auto;" /> 

**Notifications**

- **Calendar Notifications**: Enable or disable calendar alerts for attendees.
- **Email Notifications**: Enable or disable email reminders for attendees.

<img src="{{ site.baseurl }}/assets/images/eventsetting3.png" alt="Servv Event Settings" width="600" style="max-width: 100%; height: auto;" /> 

Click **Save** when editing an existing event or **Publish** when creating a new event to apply your settings.

---

## Why It Matters

Proper configuration of event settings ensures your events are accessible to the intended audience with timely notifications. By utilizing visibility controls and notification options, you enhance the attendee experience, making it more professional and seamless.
