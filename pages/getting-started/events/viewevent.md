## Navigate to the Events Page

Access the Events page from the side menu in your Servv dashboard.

 <img src="{{ site.baseurl }}/assets/images/eventsection.png" alt="Servv" width="200" style="max-width: 100%; height: auto;" />

All created events are displayed in a table with the following columns:

- Title
- Date
- Time
- Location
- Type (One-time/Recurring)
- Status

<img src="{{ site.baseurl }}/assets/images/eventlist.png" alt="Servv" width="700" style="max-width: 100%; height: auto;" />

## Use the Upcoming/Past Event Toggle

Switch between viewing upcoming and past events:

- **Upcoming Events**: Displays all events that have not yet started.
- **Past Events**: Shows all events that have concluded.

<img src="{{ site.baseurl }}/assets/images/eventsectiontoogle.png" alt="Servv" width="600" style="max-width: 100%; height: auto;" />

## Filter and Search Events

Locate specific events with ease:

- **Search by Title**: Type part of an event’s name to quickly find it.

<img src="{{ site.baseurl }}/assets/images/searchevent.png" alt="Servv" width="700" style="max-width: 100%; height: auto;" />

- **Filter by Date Range**: Use the date picker to view events within a specific time frame.

<img src="{{ site.baseurl }}/assets/images/selectdate.png" alt="Servv" width="600" style="max-width: 100%; height: auto;" />
