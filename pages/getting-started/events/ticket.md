# Ticket Type foe event

!!! info "âš¡ Premium Feature"
    The **Paid ticket** feature is available only on Premium plans.

This document provides a guide on creating different types of tickets for events. By default, the ticket type is set to "Free". To access "Paid" and "Donation" ticket types, you need to activate the **premium feature**.

### Ticket Types
**Free**: Available by default. No cost associated with the ticket.

**Paid**: A premium feature that requires activation. Allows setting a specific price for the ticket.

**Donation**: A premium feature that requires activation. Allows users to donate any amount for the ticket.

### Creating a New Ticket
Click on **Create new ticket** to open the ticket creation form.

<img src="/images/newticket1.png" alt="Servv Event Settings" width="600" style="max-width: 100%; height: auto;" />

The top section displays ticket details, which update dynamically based on the input for ticket type, name, price, and quantity.

<img src="/images/ticketdetails.png" alt="Servv Event Settings" width="600" style="max-width: 100%; height: auto;" />

**Select Ticket Type:**

- Choose between Paid, Free, or Donation.

     Note: "Paid" and "Donation" require premium feature activation.

**Name the Ticket:** 

- Enter a unique name for your ticket.

**Set Ticket Price:**

- For "Paid" type, enter the price (e.g., $5).
- For "Free" and "Donation" type, no price is required.

**Enter Ticket Quantity:**

- Specify the number of tickets available (e.g., 10).   

<img src="/images/newticket2.png" alt="Servv Event Settings" width="600" style="max-width: 100%; height: auto;" /> 

## Set Availability

- Select **Open** to make tickets available immediately after creating the event.

<img src="/images/ticketavailability.png" alt="Servv Event Settings" width="600" style="max-width: 100%; height: auto;" />

- Select **Sales start & end** to schedule a specific time/date range for ticket sales.  

<img src="/images/newticket3.png" alt="Servv Event Settings" width="600" style="max-width: 100%; height: auto;" />   

Click **Save** to create the ticket and make it available according to the selected availability settings.

Click **Cancel** to discard changes.
