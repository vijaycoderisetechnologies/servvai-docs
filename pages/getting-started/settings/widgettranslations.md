# Navigating to Servv AI Widget Translations

Follow these steps to access the `Widget Translations` section in Servv.io:

### Step 1: Click on Servv
Click on **ServvAI Event Booking** to open `Servv Ai`.

<img src="{{ site.baseurl }}/assets/images/servv.png" alt="Servv" width="250" style="max-width: 100%; height: auto;" />

## Step 2: Go to the Setting 
Click on the `Settings` tab in the dashboard menu.

<img src="{{ site.baseurl }}/assets/images/settingsection.png" alt="Servv" width="200" style="max-width: 100%; height: auto;" />

You will now see the following sections:

- `General Settings`
- `Reminders`
- `Billing Plans`
- `Fast Checkout`
- `Widget Settings`
- `Widget Translation`

## Step 3: Click Widget Translations
Open `Widget Translations`

<img src="{{ site.baseurl }}/assets/images/widgettranslationsection.png" alt="Servv" width="700" style="max-width: 100%; height: auto;" />

## Step 4: Translate widget content to different languages.

## Default Language  
`Set the primary language for the widget.` 

`Example: Hindi - हिन्दी, English`

## Language for Translation  
`Choose a language to customize widget text labels.`

`Example: English - English`

## Global Widget Translations  
Customize common labels:  

- Price Lable: `Price`  
- Filters Lable: `Filters`  
- Calendar Lable: `Calendar`  
- Price Free Lable: `Free`  
- Day label plural: `d`
- Day label singular:`d`
- Hour label plural:`h`
- Hour label singular:`h`
- Minute label plural:`m`
- Minute label singular:`m`

## Events Widget Translations  
Translate UI elements specific to events:

- Open dialog button:`Book event`
- Events list title:`Events list`
- Bundles list title:`Bundles list`
- Widget events list switch label:`Events`
- Widget bundles list switch label:`Bundles`
- Bundle add to cart button label:`Add to Cart`
- Event add to cart button label:`Add to Cart`
- Live shopping join button label:`Join`
- Live shopping start countdown:`in`
- Bundle events list title:`Included events`
- Share event panel title:`Share this event`
- Search event placeholder:`Search`
- Items counter label:`items`
- Clear filters label:`clear`
- Book button label:`Book now`
- Virtual appointment label:`Appointment`
- Virtual event label:`Virtual`
- In person event label:`In-Person`
- Webinar label:`Webinar`

- Event details button label:`Details`
- Event description field label:`Description`

## Step 5: Click on Save
Afetr making any changes clikc on `save` button.
