# Add Category Filter

### Access the Filter Dashboard
From your **Servv AI** side bar, go to Filters. This section manages all available filter types.

<img src="{{ site.baseurl }}/assets/images/filtersection.png" alt="Servv" width="200" style="max-width: 100%; height: auto;" />

### Click the Add button at the top right to begin creating a new filter then select Category.

<img src="{{ site.baseurl }}/assets/images/categoryfilter.png" alt="Servv" width="300" style="max-width: 100%; height: auto;" />

### Fill Out Category Details and save 

**Category Name**: Enter your category title — like Workshop, Art, Music, Business.

**Category Details**: Add a brief description to help your team or collaborators understand the purpose of the category 

 - **Example**: “Use this category for all marketing-related training events.”

<img src="{{ site.baseurl }}/assets/images/newcategoryform.png" alt="Category Filter Creation" width="700" style="max-width: 100%; height: auto;" />

The newly created **Category filter** will now be available as an option when creating or editing any event.

<img src="{{ site.baseurl }}/assets/images/filter1.png" alt="Servv" width="700" style="max-width: 100%; height: auto;" />
